## Deskripsi

Project ini membuat popup cookie consent sederhana menggunakan:

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Local Storage

Popup akan muncul saat pertama kali user membuka halaman.
Jika user menekan tombol "I Like Cookies", maka consent akan disimpan di localStorage dan popup tidak akan muncul kembali.
